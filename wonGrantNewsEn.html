We won a grant!

BaseALT Ltd received grant support from Innovation Promotion Fund for the development of the libdomain open library that will be used for creating and applying domain administration tools with various organizations: Samba, FreeIPA, Active Directory. Libdomain will allow to create and apply unified group policies for corporate IT infrastructure with Russian and foreign operating systems.

Moreover:

  • the library will be included into the Sisyphus repository;
  • the library development infrastructure is located in Russian Federation, owned and maintained by BaseALT Ltd;
  • developers from all countries will be able to use libdomain. For example, it will be possible to create specialized domain infrastructure management tools, applications for migration from Windows to Russian OS on Linux kernel, specialized modules for powershell.
