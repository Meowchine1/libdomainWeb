About

libdomain is an open library project that abstracts access to domain infrastructure and provides a high-level API for managing directory service objects. The project is promising in the field of import substitution in terms of migration from foreign Microsoft Active Directory domains to domestic developments of domain infrastructure.

The need to implement graphical and console applications for domain administration requires a program interface to access directory service data. The absence of such program interface requires the inclusion of all the logic for domain management in each corresponding application, and also entails multiple code duplication. Therefore, transferring the domain management logic to a separate library seems appropriate.

This development is planned to be integrated into our current projects ADMC , GPUI

This development is also planned to be used in the implementation of new projects, in particular, a module for managing Active Directory in PowerShell under Linux.
