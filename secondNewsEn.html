The project has started!On December 21, 2022, our team initiated the first stage of the project “Development of an open library for domain infrastructure management based on the Samba directory service”. As part of the stage, information will be collected and objects in the AD* domain and their properties will be analyzed for further development of the prototype system architecture.
Libdomain is a library that will simplify the interaction with LDAP by providing a simple and intuitive API, allowing its users to get integration with Qt* or GTK*. The project is developed on C with Cmake build system. You can get more information about the library code in the repository.
  • AD* — Microsoft Active Directory
  • Qt* — cross-platform interface elements library
  • GTK* — cross-platform interface elements library
